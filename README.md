# Login Site
Login / Register / Todo site with "admin panel". As admin you can delete users tasks.

## Technologies:
- React / Vite
- Bootstrap
- MySQL

## Setup:
```
git clone {link}
```
```
cd LoginSite
```
```
npm install
```
### Run:
```
npm run dev
```
